# Market Pulse India 📊
### Financial markets, explained for everyone.

A real-time cross-market financial dashboard tracking Indian & US markets, 
forex, debt markets, and unicorn valuations — built by Fabian Fernandes.

## Live Dashboard
🚀 [View Live Dashboard](https://market-pulse-india.streamlit.app)

## What it tracks
- 🇮🇳 Indian Equities — Nifty 50 + 25 Indian Giants
- 🇺🇸 US Equities — S&P 500 + 25 US Giants  
- 💱 Forex — USD/INR exchange rate
- 📈 Debt Markets — India & US 10Y Bond Yields
- 🦄 Unicorns — Top 10 Indian & US private startups

## Key Features
- Live data fetching every hour via Yahoo Finance & FRED API
- Beginner/Expert mode toggle
- Correlation heatmap across all 5 markets
- Market Mood indicator (Calm / Cautious / Panic)
- Historical market crashes timeline
- Interactive charts with hover tooltips

## Tech Stack
- Python 3.13
- Streamlit — Dashboard framework
- Plotly — Interactive charts
- pandas — Data manipulation
- yfinance — Market data
- fredapi — US Federal Reserve data

## Key Findings
1. Nifty 50 and S&P 500 are **72% correlated** — global markets move together
2. USD/INR weakened **8.5%** in 1 year — rupee at historic lows
3. Market volatility doubled in March 2026 — panic mode active
4. Rising US bond yields consistently pull Indian markets down
